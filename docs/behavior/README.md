# Диаграммы поведения системы "Тренажер для подготовки к тестов"

## Use Case Диаграмма
Описание основных сценариев использования приложения

### Акторы:
- **Студент** - основной пользователь, проходит тесты и анализирует результаты
- **Преподаватель** - создает и редактирует тесты, управляет категориями

### Основные Use Cases:
- Создание и редактирование тестов
- Добавление вопросов и управление категориями
- Прохождение тестов и проверка ответов
- Просмотр результатов и аналитика

## Диаграммы последовательности

### 1. Создание теста
Показывает процесс создания нового теста преподавателем с валидацией данных

### 2. Прохождение теста
Демонстрирует полный цикл прохождения теста студентом от начала до получения результатов

### 3. Просмотр результатов
Отображает механизм анализа результатов и прогресса обучения

### 4. Ежедневный workflow
Комплексный сценарий типичного учебного дня студента с персонализированными рекомендациями

## Используемые технологии
- **PlantUML** - для создания всех диаграмм
- **Git** - для контроля версий диаграмм
